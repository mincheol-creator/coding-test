## 문자열을 정수로 바꾸기

##### Python

##### 문제있음

```python
def solution(s):
    answer = ''
    s=s.strip()
    arr=s.split(" ")
    refreshArr=[]
    print(arr)
    for i in range(len(arr)):
        if arr[i]!='':
            refreshArr.append(arr[i])
            print(refreshArr)
    for i in range(len(refreshArr)):
        for j in range(len(refreshArr[i])):
            if j%2==0:
                answer+=refreshArr[i][j].upper()
            else:
                answer+=refreshArr[i][j].lower()
        answer+=' '
    answer=answer.strip()
    return answer
```
