## k번째 수

##### Python

```python
def solution(array, commands):
    answer = []
    temp_arr=[]
    count=0
    for i in range(len(commands)):
        min=commands[i][0]
        max=commands[i][1]
        for j in range(min-1,max):
            temp_arr.append(array[j])
        temp_arr=sorted(temp_arr)
        k=commands[i][2]
        answer.append(temp_arr[k-1])
        temp_arr=[]
    return answer
```
